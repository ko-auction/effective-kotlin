+ 챕터 담당자는 질문이 올라온것에 대해서 답변을 달아줍니다

+ 발표 내용 방식은 챕터 담당자 마음입니다!

+ 만약 업무 떄문에 바쁘다면 다른 사람과 챕터 교환을 합니다
+ 스터디날은 마이크를 켜서 적극적으로 토크토크 합니다
